# Spam List
Script to find spam comments on lbry

# Usage

`git clone https://github.com/lpessin/spamlist_lbry.git`

`pip install requests`

 `pip install xlsxwriter`
 
 `cd spam_list_lbry`
 
 `python script.py`

Requires SDK or LBRY instance running

# Settings
Edit script.py
